# Hi, I'm Sena Çakıcı 🔬

I'm a Data Scientist working at the intersection of **machine learning**, **scientific computing**, and **real-world engineering problems**. My work spans from physics-informed models to production-ready ML pipelines.

---

##  What I Work On

I'm particularly interested in applying machine learning to domains where the underlying physics or domain knowledge matters not just fitting a curve, but building models that *understand* the system they're modeling.

My projects range from **computational chemistry** and **materials science** to **MLOps**, **uncertainty quantification**, and **anomaly detection**. I enjoy problems where the data is messy, the stakes are real, and a clever model can make a genuine difference.

---

##  Selected Projects

### 🧪 Physics-Informed & Scientific ML

**[uncertainty-drug-binding](https://github.com/senacakici/uncertainty-drug-binding)** *(Private)*
Physics-informed ML model with uncertainty quantification for drug-target binding affinity prediction. Combines domain knowledge with probabilistic modeling to produce trustworthy predictions in high-stakes settings.

**[DFT_NEB_ML-pesH2-Ni](https://github.com/senacakici/DFT_NEB_ML-pesH2-Ni)**
DFT and ML-based approach for studying H₂ adsorption on Ni(111) surfaces — potential energy surface construction using first-principles simulations combined with machine learning surrogates.

**[ml-pes-project](https://github.com/senacakici/ml-pes-project-)**
ML-based potential energy surface estimation. Learning molecular energy landscapes from data.

**[Scatteringnet](https://github.com/senacakici/Scatteringnet)**
Neural network model for predicting particle radius from scattering data. Applied deep learning to inverse problems in experimental physics.

**[FibreBarrier-ML](https://github.com/senacakici/FibreBarrier-ML)**
Physics-informed ML model for predicting water vapor transmission rate (WVTR) of coated fibre-based sustainable packaging materials.

**[mfp-thermal-emt](https://github.com/senacakici/mfp-thermal-emt)**
EMT-based Python tool for thermal conductivity modelling of porous moulded fibre products and polymer barriers.

### ⚙️ MLOps & Production ML

**[workflow-intelligence](https://github.com/senacakici/workflow-intelligence)**
Task classification and workload anomaly detection using weak supervision. Built with scikit-learn, served via FastAPI, with a Streamlit interface. An end-to-end ML system designed for real operational environments.

**[FMCG-Sales-Lakehouse-Pipeline](https://github.com/senacakici/FMCG-Sales-Lakehouse-Pipeline)**
A data lakehouse pipeline for FMCG sales analytics. Covers ingestion, transformation, and analytical layer design at scale.
